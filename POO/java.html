Sintaxe e Fundamentos em Java

Nesta seção, vamos abordar os conceitos básicos da linguagem Java, incluindo variáveis, tipos de dados, operadores, estruturas de controle, funções e mais


Variáveis em Java



  • Variável Local:

    São declaradas dentro de um método ou bloco de código.

    São criadas quando o método é chamado e destruídas quando o método ou bloco é concluído.

    Podem ser acessadas apenas dentro do método.

    Não têm valor padrão e devem ser inicializadas antes de serem usadas.


  • Variáveis de instância:

    • São declaradas dentro de uma classe, mas fora de qualquer método ou bloco.

    São criadas quando um objeto da classe é instanciado e destruídas quando o objeto é coletado pelo lixo.

    Podem ser acessadas por qualquer método da classe.

    Têm valor padrão (0 para números, false para booleanos, null para objetos, etc.) se não forem inicializadas explicitamente.


  • Variáveis Estáticas:

    São declaradas com a palavra-chave static dentro de uma classe.

    São criadas quando a classe é carregada e destruídas quando a classe é descarregada.

    Podem ser acessadas por qualquer método da classe, sem a necessidade de criar um objeto da classe.

    Têm valor padrão (0 para números, false para booleanos, null para objetos, etc.) se não forem inicializadas explicitamente.


// Variável local
public class Exemplo {
    public static void main(String[] args) {
        int x = 10; // Variável local
        System.out.println(x);
    }
}

// Variável de instância
public class Exemplo {
    int x =  10;  //Variável de instância

    public static void main(String[] args) {
        Exemplo exemplo = new Exemplo();
        System.out.println(exemplo.x);
    }
}

// Variável estática
public class Exemplo {
    static int x =  10; // Variável estática

    public static void main(String[] args) {
        System.out.println(Exemplo.x);
    }
}


Tipos de Dados em Java

Java é uma linguagem estaticamente tipada, o que significa que o tipo de dado de uma variável é conhecido em tempo de compilação. Os principais tipos de dados em Java são:

  • Primitivos:

    Inteiros: 'Byte', 'Short', 'Int', 'Long'

    Ponto flutuante: 'Float' , 'Double'

    Booleano: 'Boolean'

    Caractere: 'char'

    Referência: 'String', 'Array', 'Classe', 'Interfaces'


        
// inteiro
int x = 10;

// ponto flutuante
float y = 10.5;

// booleano
booelan w = true;

// caractere
char z = "A";

// Referência
String nome = "Fêmea";

Operadores em Java

Os operadores em Java são usados para realizar operações aritméticas, de comparação, lógicas, de atribuição, etc. Os principais operadores em Java são:

Operadores aritméticos:

+ (adicão), - (subtração), * (multiplicação), / (divisão), % (resto da divisão)

Operadores aritméticos em Java podem ser usados em variáveis de instância ou de classe.

Comparação:

== (igual a), != (diferente de), < (menor que), <= (menor ou igual a), > (maior que), >= (maior ou igual a)

Operadores de comparação em Java podem ser usados em variáveis de instância ou de classe.

Lógicos:

&& (e), || (ou), ! (não)

Operadores lógicos em Java podem ser usados em variáveis de instância ou de classe.

Atribuição:

= (atribuição), += (adiciona), -= (subtrai), *= (multiplica), /= (divide), %= (resto da divisão)

Operadores de atribuição em Java podem ser usados em variáveis de instância ou de classe.




// Aritméticos
int x = 10;
int y = 5;
int soma = x + y;
System.out.println(soma);

// Comparação
int x = 10;
int y = 5;
boolean igual = x == y;
System.out.println(igual);

// Lógicos
boolean x = true;
boolean y = false;
boolean e = x && y;
System.out.println(e);

// Atribuição
int x = 10;
x += 5;
System.out.println(x);

Estruturas de Controle em Java

IF-ELSE

Usada para tomar decisões com base em condições 
// If-else
int x =     10;
if (x >  5) {
    System.out.println("x é maior que 5 ");
} else {
    System.out.println("x é menor ou igual à 10 ");
}

FOR

O laço for é usado para executar um bloco de código repetidamente por um número determinado de vezes.

// For
for (int x = 0; x 10; x++) {
    System.out.println(x);
}

WHILE

O laço while é usado para executar um bloco de código enquanto uma condição for verdadeira.

// While
int x = 0;
while (x < 10) {
    System.out.println(x);
    x++;
}

DO-WHILE

O laço do-while é usado para executar um bloco de código enquanto uma condição for verdadeira. A diferença entre o while e o do-while é que o do-while executa o bloco de código pelo menos uma vez antes de verificar a condição.

// Do-while
int x = 0;
do {
    System.out.println(x);
    x++;
} while (x < 10);

Funções

As funções em Java são blocos de código que podem ser chamados várias vezes a partir de diferentes partes do programa. As funções podem receber parâmetros e retornar valores.

// Função
int soma(int x, int y) {
    return x + y;
}

int resultado = soma(10, 5);

Parâmetros e Retorno de Funções

As funções em Java podem receber parâmetros e retornar valores. Os parâmetros são valores que são passados para a função quando ela é chamada, e o retorno é o valor que a função devolve ao final de sua execução.

Sobrecarga de Funções

A sobrecarga de funções em Java é a capacidade de criar várias funções com o mesmo nome, mas com parâmetros diferentes. Isso permite que as funções sejam mais flexíveis e possam ser usadas em diferentes contextos.

// Sobrecarga de funções
public class Exemplo {
    public static void main(String[] args) {
        int resultado1 = soma(2, 3 );
        System.out.println(resultado1);

        int resultado2 = soma(2, 3, 4);
        System.out.println(resultado2);
    }

    public static int soma(int a, int b) {
        return a + b;
    }

    public static int soma(int a, int b, int c) {
        return a + b + c;
    }
}

Recursividade

A recursividade em Java é a capacidade de uma função chamar a si mesma. Isso permite que as funções sejam mais concisas e possam ser usadas para resolver problemas complexos.

// Recursividade
public class Exemplo {
    public static void main(String[] args) {
        int resultado = fatorial(5);
        System.out.println(resultado);
    }

    public static int fatorial(int n) {
        if (n == 1) {
            return 1;
        } else {
            return n * fatorial(n - 1);
        }
    }
}
